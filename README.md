# PCR-Primer-Design-for-Human-ACTB-Gene
This repository contains the design and validation of PCR primers for the ACTB (Beta-actin) gene from Homo sapiens. The primers were designed as part of a practical assignment focused on understanding primer specificity, target enrichment, and PCR fundamentals using NCBI Primer-BLAST
Author : Satyam choudhary 

# Gene Information
Gene Name: ACTB (Beta-actin)

Organism: Homo sapiens

Reference mRNA: NM_002046.7

Gene Type: Housekeeping gene


# Primer Design Tool

Tool Used: NCBI Primer-BLAST

Database for Specificity Check: Human Genome (taxid:9606) 



# Primer Design Parameters

The primers were designed using the following standard parameters:

Primer length: 18–22 bp

Melting temperature (Tm): ~60 °C

GC content: 40–60%

PCR product size: 80–200 bp

Max Tm difference between primers: ≤ 2 °C
